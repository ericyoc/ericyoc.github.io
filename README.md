## About

This is the source repository for [ericyocam.com](https://ericyocam.com), a static HTML/CSS personal site hosted via GitHub Pages with a custom domain managed through Cloudflare.

The site serves as a comprehensive academic and professional profile covering:

- 🎓 Education — 7 degrees across cybersecurity, computer engineering, computer science, business administration, and finance
- 🏫 Academic Appointments — 7 appointments across doctoral, graduate, and undergraduate institutions (2007–present)
- 💼 Professional Experience — 25+ years in industry including T-Mobile, Microsoft, Apple, HP, and Intuit
- 📄 Publications — 20+ peer-reviewed publications in IEEE Access, IEEE Transactions, Springer, and arXiv
- 🔬 Patents — 23+ issued US patents assigned to T-Mobile USA
- 🏆 Honors & Awards — IEEE Senior Member, DoD X-Force Fellow, graduate fellowships
- 🛠️ Service — ABET evaluator, CISSP/CRISC/CDPSE cert author, peer reviewer, mentor

## Tech Stack

| Layer | Technology |
|---|---|
| Structure | HTML5 |
| Styling | CSS3 (custom properties, CSS Grid, Flexbox) |
| Fonts | Google Fonts — Playfair Display, Source Sans 3 |
| Icons | Font Awesome 6.5 |
| Hosting | GitHub Pages |
| DNS | Cloudflare (DNS Only, no proxy) |
| HTTPS | Let's Encrypt via GitHub Pages |

No JavaScript frameworks, no build tools, no dependencies — just a single self-contained `index.html`.

## Local Development

No build step required. Just open the file directly:

```bash
# Clone the repo
git clone https://github.com/ericyoc/ericyoc.github.io.git
cd ericyoc.github.io

# Open in browser
open index.html
```

Or serve locally with Python:

```bash
python3 -m http.server 8000
# Visit http://localhost:8000
```

---

## Deployment

Every push to `main` automatically deploys to [ericyocam.com](https://ericyocam.com) via GitHub Pages.

```bash
git add .
git commit -m "your update message"
git push origin main
```

DNS is managed through Cloudflare with the following configuration:

| Type | Name | Target | Proxy |
|---|---|---|---|
| A | @ | 185.199.108–111.153 | DNS Only |
| CNAME | www | ericyoc.github.io | DNS Only |
